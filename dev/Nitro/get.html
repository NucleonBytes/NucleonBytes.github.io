FAQ //How often are bug patches uploaded?

We aim to push bug fixes as soon as we patch bugs to reduce the chances of vulnerabilities.


FAQ //How often should i check this page for updates?

Until we reach the milestone at v2, we highly recommend checking this page at least every two weeks. Once we reach v2, updates will be automatic for both client and developer.


FAQ //Will future updates break current API?

We aim to make updates backward compatible, however, in some cases it may not be possible to keep backwards compatibility with older API (e.g. security) If this happens, application made with versioning API will be automatically converted to latest API.


FAQ //How do i enable API versioning?

To enable versioning to force backwards compatibility, set runtime>version in the manifest to the API version you want to force. Note: we do not recommend forcing backwards compatibility as it may void some security updates.


FAQ //Why wont my app render?

If you have broken code which could potentially break your whole app or cause security issues, NitrixCore will not render your app. To fix this, make sure there are no errors in your syntax.
