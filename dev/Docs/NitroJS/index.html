var nitro = new NitroCore();
NitroCore is the core engine behind the multi-engine processor. Before you can start using the built-in components of the engine, you must create a new instance of the library.
nitro.addPermission('app');
When an instance of NitroCore has been initialized, you will need to grant access to your app - so you can query the app module for information about your app and its runtime.
app.get('appName');
Once you have been granted access to the app module, you may query it for information such as application name. (Note this is different from ApplicationName - which returns the namespace of your runtime)
myApplication
